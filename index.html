HTML Tables

Table Tags

Tag Name Description Structure
<table> Table Element The wrapper element for all HTML tables and has content that is used to represent a two-dimensional table made of rows and columns.
<thead> Table Head Element The set of rows defining the column headers in a table.
<tbody> Table Body Element It is a semantic element that will contain all table data other than table heading and table footer content.
<tr> Table Row Element The table row contaner.
<td> Table Data Element The table cell container.
<th> Table Heading Element is used to add titles to rows and columns of a table and must be enclosed in a table row element, <tr>.
<tfoot> Table Footer Element The set of rows defining the footer in a table.
HTML tables allow web developers to arrange data into rows and columns.

HTML Elements

Element Tags

An HTML element is a piece of content in an HTML document and uses the following syntax: opening tag (<) + content + closing tag (/>).

Tag Name Description Structure
<html> HTML Element The root of an HTML document, should be added after the !DOCTYPE declaration. All content/structure for an HTML document should be contained between the opening and closing <html> tags.
<head> Head Element Contains general information about an HTML page that isn’t displayed on the page itself. This information is called metadata and includes things like the title of the HTML document and links to stylesheets.
<title> Title Element contains a text that defines the title of an HTML document. The title is displayed in the browser’s title bar or tab in which the HTML page is displayed. The <title> element can only be contained inside a document's <head> element.
<body> Body Element Represents the content of an HTML document.
<h1>-<h6> Heading Elements The heading elements are ordered from the highest level <h1> to the lowest level <h6>.
<div> Div Element Is used as a container that divides an HTML doucments into sections.
<p> Paragraph Element Contains and display a block of text.
<span> Span Element Is an inline container for text and can be used to group text for styling purposes. <p><span>This text</span> may be styled differently than the surrounding text.</p>
<em> Emphasis Element Emphasizes text and browsers will usually italicize the emphasized text by default. <p>This<em>word</em> will be emphasized in italics.</p>
<strong> Strong Element Highlights important, serious, or urgent text and browsers will normally render this highlighted text in bold by default. <p>This is<strong>important</strong> text!.</p>
<br> Line break Element Will create a line break in text. It requires only an opening tag.
<ul> Unordered List Element Is used to create a list of items in no particular order. Each individual list item will have a bullet point by default.
<ol> Ordered List Element class="break-td"Creates a list of items in sequential order. Each list item appears numbered by default.
<li> List Item Creates list items inside Ordered lists <ol> and Unordered lists <ul>.
<img> Image Element Embed images in documents. The src attribute contains the image URL and is mandatory. <img> is an empty element meaning it should not have a closing tag. <img src="path/to/image" >
<video> Video Element Embeds a media player for video playback. The src attribut will containe the URL to the video. <video src="test-video.mp4" controls>
Video not supported </video>
<anchor> Anchor Element Is used to create hyperlinks in an HTML document. The hyperlinks can point to other webpages, files on the same server, a location on the same page, or any other URL via the hyperlink reference attribute, href. The href determines the location the anchor element points to. <a href="http:////www.codecademy.com" > Visit this site </a>
The content of an HTML element is the information between the opening and closing tags of an element.
